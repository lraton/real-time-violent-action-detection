# Addestramento del Modello (`train_model.py`)
   Serve per addestrare il modello per riconoscere coltelli preso da roboflow
   Una volta compeltato lo script bisogna eseguire questo comando: per addestrarlo
   ```
   yolo detect train data="knife-detection-1/data.yaml" model=yolo11n.pt epochs=50 imgsz=640
   ```
